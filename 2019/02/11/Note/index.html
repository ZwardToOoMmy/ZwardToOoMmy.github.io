2019寒假集训题

D.约数个数和

题面

设d(x)为x的约数个数，给定N、M，求 $\sum_{i=1}^{N}\sum_{j=1}^{M}d(i×j)$.

Input

输入文件包含多组测试数据。

第一行，一个整数T，表示测试数据的组数。

接下来的T行，每行两个整数N、M。

Output

T行，每行一个整数，表示你所求的答案。

Sample Input

2
7 4
5 6

Sample Output

110

121

Hint

$1\leq N,M \leq 50,000$

$1\leq T \leq 50,000$
